 Code example

>My project<

Task:

Using prompt, request any number from the user and check it according to the following parameters: more/less than 100; even/odd; fractional/non; fractional; positive/negative.

Display the test results on the screen in any available way in approximately the following format - "The number you entered is less than one hundred, even, positive, not fractional."

Decision:

let res_1, res_2, res_3, res_4;
const userNumber = prompt("Enter any number:");
(+userNumber) || userNumber && alert("Please enter a number!");
res_1 =
(userNumber > 100 && "more than a hundred") ||
(userNumber <100 && "less than a hundred") ||
(userNumber == 100 && "equals 100");
res_2 =
(userNumber > 0 && "positive") || (userNumber <0 && "negative");
res_3 =(userNumber % 1 == 0 && "whole") || (userNumber % 1 !== 0 && "fractional");
res_4 =
(userNumber % 2 == 0 && "even") || (userNumber % 2 !== 0 && "odd");
alert(`The number you entered ${res_1}, ${res_2}, ${res_3}, ${res_4}`);
